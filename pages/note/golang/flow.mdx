# 流程控制

## if

```go
if 条件 {
    // do something
} else if 条件 {
    // do something
} else {
    // do something
}
```

## switch

```go
switch 变量 {
case 值1:
    // do something
case 值2:
    // do something
default:
    // do something
}
```

注意，与其他语言不同，Go 中的 `switch` 语句不需要 `break`，一旦条件符合自动终止。

但是如果需要继续执行下一个条件，可以使用 `fallthrough` 关键字。

```go
switch 变量 {
case 值1:
    // do something
    fallthrough
case 值2:
    // do something
default:
    // do something
}
```

## for

Go 中只提供了 `for` 循环，没有 `while` 循环。

但是提供了三种 `for` 循环的写法。

## 无限循环

```go
for {
    // do something
}
```

例如：

```go
for {
    fmt.Println("Hello, World!")
}
```

如果想要终止循环，可以使用 `break` 关键字。

```go
for {
    fmt.Println("Hello, World!")
    break
}
```

## 条件循环

```go
for 条件 {
    // do something
}
```

例如：

```go
var i int = 0

for i < 10 {
    fmt.Println(i)
    i++
}
```

## 计数循环

```go
for 初始语句; 条件; 结束语句 {
    // do something
}
```

例如：

```go
for i := 0; i < 10; i++ {
    fmt.Println(i)
}
```

## 循环中的控制语句

`break`：终止循环。

`continue`：跳过当前循环，继续下一次循环。

```go
for i := 0; i < 10; i++ {
    if i == 5 {
        continue
    }
    fmt.Println(i)
}
```

表示当 `i` 等于 5 时，跳过当前循环，继续下一次循环。

## label 与 goto

Go 中可以使用 `label` 和 `goto` 来实现跳转。

```go
label:
    // do something
goto label
```

例如：

```go
var i int = 0

label:
    fmt.Println(i)
    i++
    if i < 10 {
        goto label
    }
```
表示当 `i` 小于 10 时，跳转到 `label` 标签处。
